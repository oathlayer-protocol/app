# Mini App Module

World Mini App for provider registration and claim filing via World App.

## Purpose

Mobile-first interface for World App users to:
- Register as SLA providers (World ID verification built into Mini App SDK)
- File claims against SLAs via MiniKit `sendTransaction`

## Stack

- Next.js + World Mini App SDK (MiniKit)
- Targets World Chain Sepolia (4801)

## Commands

```bash
npm install
npm run dev
```
